#### SeSReportFile

Used to attach a file to the report.

```javascript

```


**Parameters:**

|  **Name** | **Type** | **Description** |
| ---------- | -------- | --------------- |
| path | string |  Path to a file. |
| caption | string |  Caption to show in the report. |
