#### DoSelectItem

Selects an item with the specified name.

```javascript
SeS('SomeAndroidSpinner').DoSelectItem(name)
```


**Parameters:**

|  **Name** | **Type** | **Description** |
| ---------- | -------- | --------------- |
| name | number |  Item name. |




**Returns:**

boolean: 'true' if success, 'false' otherwise.
